# Chapter 05: The simple linear regression model

Companion materials for *Regression with R and Python: Description, Prediction, and Causal Analysis for Applied Social Science and Econometrics*.

## R Markdown workbooks

- `chapter05_workbook.Rmd` is the reference workbook that shows chapter code in textbook order.
- `chapter05_runnable_workbook.Rmd` is the runnable workbook with setup code added so the chapter can usually be executed end-to-end.

## R code boxes

- Inference for regression in R
- Robust inference in R

## Python code boxes

- Inference for regression in Python
- Robust inference in Python
- Data generating process and Monte Carlo simulations in Python

## Notes

- Scripts mirror the current textbook code blocks in the order they appear in the book.
- The repository is aimed at advanced undergraduate and beginning master's students in applied social sciences and econometrics.
- Relative file paths are left as printed in the textbook source.
