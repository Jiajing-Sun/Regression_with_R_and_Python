# Chapter 06: Multiple Linear Regression

Companion materials for *Regression with R and Python: Description, Prediction, and Causal Analysis for Applied Social Science and Econometrics*.

## R Markdown workbooks

- `chapter06_workbook.Rmd` is the reference workbook that shows chapter code in textbook order.
- `chapter06_runnable_workbook.Rmd` is the runnable workbook with setup code added so the chapter can usually be executed end-to-end.

## R code boxes

- Multiple regression in R
- $F$-test in R
- Robust $F$-test in R
- Confidence intervals for expected value and prediction intervals in R

## Python code boxes

- Multiple regression in Python
- $F$-test in Python
- Robust $F$-test in Python
- Confidence intervals for expected value and prediction intervals in Python

## Notes

- Scripts mirror the current textbook code blocks in the order they appear in the book.
- The repository is aimed at advanced undergraduate and beginning master's students in applied social sciences and econometrics.
- Relative file paths are left as printed in the textbook source.
