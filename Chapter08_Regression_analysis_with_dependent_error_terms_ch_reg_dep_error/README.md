# Chapter 08: Regression analysis with dependent error terms

Companion materials for *Regression with R and Python: Description, Prediction, and Causal Analysis for Applied Social Science and Econometrics*.

## R Markdown workbooks

- `chapter08_workbook.Rmd` is the reference workbook that shows chapter code in textbook order.
- `chapter08_runnable_workbook.Rmd` is the runnable workbook with setup code added so the chapter can usually be executed end-to-end.

## R code boxes

- Project Star and clustering in R
- Project Star and multilevel models in R
- The within estimator with municipal data in R

## Python code boxes

- Project Star and clustering in Python
- Project Star and multilevel models in Python
- The within estimator with municipal data in Python

## Notes

- Scripts mirror the current textbook code blocks in the order they appear in the book.
- The repository is aimed at advanced undergraduate and beginning master's students in applied social sciences and econometrics.
- Relative file paths are left as printed in the textbook source.
