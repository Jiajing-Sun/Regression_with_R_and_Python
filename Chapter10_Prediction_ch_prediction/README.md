# Chapter 10: Prediction and Nonparametric regression

Companion materials for *Regression with R and Python: Description, Prediction, and Causal Analysis for Applied Social Science and Econometrics*.

## R Markdown workbooks

- `chapter10_workbook.Rmd` is the reference workbook that shows chapter code in textbook order.
- `chapter10_runnable_workbook.Rmd` is the runnable workbook with setup code added so the chapter can usually be executed end-to-end.

## R code boxes

- Example with training and test data
- Cross-validation in R
- Ridge and lasso in R
- Regression tree in R
- Comparison of the different prediction models

## Python code boxes

- Example with training and test data in Python
- Cross-validation in Python
- Ridge and lasso in Python
- Regression tree in Python
- Comparison of the different prediction models in Python

## Notes

- Scripts mirror the current textbook code blocks in the order they appear in the book.
- The repository is aimed at advanced undergraduate and beginning master's students in applied social sciences and econometrics.
- Relative file paths are left as printed in the textbook source.
